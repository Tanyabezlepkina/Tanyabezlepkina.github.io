## Что такое Git?

**Git** — это распределённая система управления версиями, которая позволяет отслеживать изменения в файлах и координировать работу над проектами между несколькими разработчиками.

### Основные понятия Git

| Понятие | Описание |
|---------|----------|
| **Репозиторий** | Хранилище файлов и истории изменений |
| **Коммит** | Снимок состояния файлов в определённый момент |
| **Ветка** | Отдельная линия разработки |
| **Удалённый репозиторий** | Репозиторий на сервере (например, GitHub) |
| **Merge** | Объединение изменений из разных веток |

### Основные команды Git

```bash
# Инициализация репозитория
git init

# Добавление файлов в индекс
git add file.txt
git add .          # добавить все файлы

# Создание коммита
git commit -m "Сообщение о изменениях"

# Отправка изменений на удалённый репозиторий
git push

# Получение изменений из удалённого репозитория
git pull

# Просмотр статуса
git status

# Просмотр истории коммитов
git log --oneline
