## Что такое Markdown

Markdown — это лёгкий язык разметки для создания форматированного текста.

### Основные возможности

- **Заголовки** — `# H1`, `## H2`
- **Жирный текст** и *курсив*
- **Списки** (маркированные и нумерованные)
- [Ссылки](https://example.com)
- `Код`

Markdown используется везде: на GitHub, в Hugo, в Obsidian.
